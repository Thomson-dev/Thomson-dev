## 🚧 Projects

### 🍔 Multi-Vendor Food Ordering App *(Flutter)*
A full-featured food ordering platform supporting customer, vendor, rider, and admin roles. Built with **Flutter + Provider** for state management, **Socket.IO** for real-time order tracking, and **OpenStreetMap** for live delivery maps — no paid APIs.

### 💬 Real-Time Chat App *(Node.js + Socket.IO)*
A WebSocket-powered chat application with room-based messaging and live user presence. Deployed on Railway.

### 📚 Library Management System *(Spring Boot)*
A REST API for managing books, members, and borrow records. Built on Spring Boot with **JPA/Hibernate**, **PostgreSQL**, and a layered architecture (entities → repositories → services → controllers → DTOs).

### 🔍 Multithreaded SERP Analyzer *(Java)*
A concurrent web scraping tool leveraging Java multithreading to parse search engine result pages in parallel — demonstrating thread safety, `ReentrantLock`, and performance optimisation.

### 📡 Light Tracker API *(Node.js)*
A REST API for tracking and logging light intensity sensor data over time, designed with clean endpoint structure and MongoDB persistence.
